Latihan HTML

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat dan mendesain halaman web. HTML menyediakan struktur dasar untuk konten web, yang kemudian dapat diatur dan diperindah dengan CSS (Cascading Style Sheets) dan diberi interaktivitas dengan JavaScript.
